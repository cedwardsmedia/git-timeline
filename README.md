__Git Timeline__ is a simple git log viewer utilizing [Vertical Timeline](https://codyhouse.co/gem/vertical-timeline/) by [CodyHouse](https://codyhouse.co) and PHP for the git work.

## Requirements
 - PHP 5
 - Git
